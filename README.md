# Home Network Documentation

## Pi-hole Setup
DNS-based ad blocker running on Raspberry Pi Zero 2 W. Blocks ads network-wide for all connected devices using Cloudflare DNS.

## Pi-hole with Tailscale Integration
Extends Pi-hole functionality to work remotely via Tailscale VPN. Enables ad-blocking on any device from anywhere without exposing Pi-hole to the internet.

## Tailscale Exit Node with Security Hardening
Configures Ubuntu home server as a secure Tailscale exit node. Implements UFW firewall rules to restrict public access while allowing local network and Tailscale connections. Enables secure internet browsing through home network when using public Wi-Fi.
